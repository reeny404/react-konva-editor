### 기능 요약

- 다각형(Polygon)을 Konva `Line`(closed) 속성으로 렌더링하여 캔버스에 배치
- 상단 버튼을 통해 두 가지 방식으로 다각형 생성 지원
  - **점 기반 polygon**: 캔버스를 클릭하며 자유롭게 다각형의 꼭지점을 추가하고 완성
  - **rect 기반 polygon**: 드래그 앤 드롭으로 사각형 영역을 지정해 특정 패턴의 다각형 생성 (구현 확장에 용이)
- 다각형 선택 및 편집 모드 지원 (꼭지점 추가/상세 이동/삭제)

### 조작법

- **점 기반 다각형 그리기 (`점 기반 polygon` 툴 활성화 시)**: 
  - 바닥(배경) 좌클릭으로 첫 꼭지점과 이후 꼭지점을 쭉 추가합니다.
  - 마우스를 이동하면 다음 꼭지점이 연결될 위치를 미리 보여줍니다(`previewPoint`).
  - 꼭지점이 3개 이상 모였을 때 시작점을 클릭하면 다각형 그리기가 완료됩니다.
- **Rect 기반 다각형 그리기 (`rect 기반 polygon` 툴 활성화 시)**: 
  - 캔버스를 쭉 드래그한 후 마우스를 놓으면 지정한 영역(임시 가이드 사각형 기준)에 다각형이 생성됩니다.
- **선택 해제**: 바닥(배경) 좌클릭
- **선택**: 캔버스에 생성된 다각형 도형/선 클릭
- **편집 모드**: 다각형을 **더블 클릭**하면 꼭지점을 세밀하게 다룰 수 있는 편집 모드로 진입합니다. (현재 선택 지우기 시 해제됨)
- **꼭지점 조작 (편집 모드에서만 동작)**:
  - **이동**: 꼭지점 원(흰색)을 드래그
  - **추가**: 다각형의 모서리 선분을 클릭
  - **삭제**: 꼭지점 원(흰색)을 우클릭

### 구현 포인트

#### 다각형 도구 상태 및 생성/편집 액션 (Store & Command)

- `polygonToolStore.ts`를 사용해 현재 활성화된 툴 상태, 다각형을 그리는 도중의 미리보기(`draft`), 그리고 현재 캔버스 상호작용 모드(`drawing`, `editing` 등)를 추적합니다.
- `polygonCommands.ts`에 다각형 시작, 완료, 점 추가 및 수정/삭제 등의 액션을 모아 유저 입력 이벤트를 처리합니다.

#### 다각형 렌더링과 편집 캔버스 (Canvas)

- Konva의 기본 노드인 `Line` 컴포넌츠에 `closed=true`를 주어 단일한 다각형을 표현합니다.
- 편집 모드(`isEditing` 상태)일 경우 사용자 경험을 높이기 위해 다음의 2가지를 함께 렌더링합니다:
  - **숨겨진 Hit Area 모서리**: 보이지 않는(투명한 수준의) 두꺼운 `Line`을 통해 클릭 범위를 넓히고, 클릭 시 해당 선분 인덱스 기반으로 `insertVertex`가 작동해 점이 추가됩니다.
  - **편집 꼭지점**: `Circle` 도형을 올려 `draggable`로 점 좌표를 변경(`moveVertex`)하거나 이벤트를 방지하고 삭제(`removeVertex`)합니다.
- 점 기반으로 그리는 중일 때(`draft`) 시작점에 마우스를 올릴 시 `hitStrokeWidth`를 넓혀두었고, 초록색으로 칠해져 도형을 닫을 수 있음을 시각적으로 표시(`isHoveringStartPoint`)합니다.

### 파일 구성

- `Canvas.tsx`: 점/rect 기반 도구 스위치 버튼, 다각형 렌더링 로직, 이벤트 핸들링 (진행중인 모양 및 편집 UI 포함)
- `stores/polygonToolStore.ts`: 현재 다각형 생성 및 수정 진행 상황과 툴 종류 상태를 전역 저장
- `commands/polygonCommands.ts`: 다각형 관련 도구 전환 및 캔버스 입력/객체 수정에 대응하는 데이터 액션 분리
- `types/`: `PolygonPoint` 등 다각형 기능 중심의 인터페이스 타입 정의 보관
- `utils/`: 다각형 점의 평탄화 등을 처리할 유틸 지원 (폴더 내 존재)
